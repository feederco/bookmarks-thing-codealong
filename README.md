# BookmarksThing

This is the code used for the Technigo Node.js codealong held on 2021-12-17.

## License:

Do whatever you want!

## Prerequisites

- Install Mongodb

## Running

```shell
git clone git@github.com:feederco/bookmarks-thing-codealong.git
cd bookmarks-thing-codealong
```

### Run frontend

```shell
cd frontend
npm install
npm start
```

### Run backend

```shell
cd backend
npm install
npm start
```
